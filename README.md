# 合格マスター

高卒認定・大学入試・東大入試に対応した、インタラクティブなHTML学習教材です。

- **完全に静的**（サーバー不要）。`index.html` を開くだけで動作します。
- 学習の進捗・成績・弱点は各ブラウザの `localStorage` に保存されます（端末外へは送信されません）。
- ダークモード対応、キーボード操作（1〜4キーで解答・Enterで次へ）対応。

## コース構成

| セクション | コース | 内容 |
|---|---|---|
| 1 | 高卒認定マスター | 全科目の基礎（高等学校卒業程度認定試験） |
| 2 | 大学入試マスター | 高3までの範囲・標準〜中堅レベル |
| 3 | 東大入試マスター | 記述・論述・思考力 |

## ローカルでの表示

ファイルをブラウザで直接開くか、簡易サーバーを使います。

```bash
# Python があれば
python -m http.server 8000
# → http://localhost:8000 を開く
```

## ファイル構成

- `index.html` — 画面の骨組み
- `style.css` — デザイン（ライト/ダーク）
- `data1.js` / `data2.js` — 高卒認定コースの教材データ
- `data-daigaku.js` — 大学入試コースの教材データ
- `data-toudai.js` — 東大入試コースの教材データ
- `app.js` — 画面遷移・クイズ・進捗保存のロジック

## 注意

試験日程・出願方法・入試制度・出題範囲は変わることがあります。最新情報は各公式（文部科学省・各大学）で必ず確認してください。
